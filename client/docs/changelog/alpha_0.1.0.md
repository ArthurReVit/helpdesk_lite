# Alpha 0.1.0 - Initial Client Setup

## Features

- Initial project structure using Vite and React 19.
- Redux Toolkit and RTK Query integration.
- Authentication system:
  - Login page implemented.
  - JWT token handling via cookies.
  - `authApi` with login and current user (`me`) endpoints.
- Basic routing with React Router 7.
- Dashboard skeleton for authenticated users.

## Technical Changes

- Configured `tsconfig` for modern React/TypeScript features.
- Implemented `authCookies` for persistent authentication.
- Set up base query logic for API communication.
