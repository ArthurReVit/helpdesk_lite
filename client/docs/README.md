# Helpdesk Lite - Client Documentation

## Description

Helpdesk Lite Client is the frontend interface for the Helpdesk Lite application. It provides a modern, responsive user experience for managing support tickets, interacting with the backend API, and maintaining user sessions.

## Tech Stack

- **React 19**: Core UI framework.
- **Vite**: Build tool and development server.
- **Redux Toolkit**: State management.
- **RTK Query**: Data fetching and caching.
- **React Router 7**: Client-side routing.
- **TypeScript**: Static typing for enhanced developer experience.
