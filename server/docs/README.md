# Helpdesk Lite - Server Documentation

## Description

Helpdesk Lite is a web application that provides a platform for users to create and manage support tickets. It is a lightweight and easy-to-use application that can be used by individuals or small teams to manage their support tickets.

## Tech Stack

- Python 3.12
- FastAPI
- SQLAlchemy
- PostgreSQL
